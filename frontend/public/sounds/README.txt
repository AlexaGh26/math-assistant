¡IMPORTANTE! Archivos de Sonido para Mateo Matemático

Por favor agrega los siguientes archivos de sonido en este directorio:

1. add.mp3     - Sonido para adición de números
2. clap.mp3    - Sonido de aplauso para celebración
3. count.mp3   - Sonido al contar elementos
4. negative.mp3 - Sonido especial para números negativos
5. pop.mp3     - Sonido CORTO de notificación (IMPORTANTE: debe ser un único sonido breve de 0.5-1 segundo, NO una canción)
6. subtract.mp3 - Sonido para la resta
7. success.mp3 - Sonido de éxito al finalizar una animación
8. whoosh.mp3  - Sonido para transiciones

Para más detalles, consulta frontend/README-AUDIO.md

Nota: Si no se proporcionan estos archivos, la aplicación funcionará correctamente
pero los efectos de sonido no estarán disponibles.
